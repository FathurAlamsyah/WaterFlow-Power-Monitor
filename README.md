# WaterFlowPowerMonitor

WaterFlowPowerMonitor is an IoT-based monitoring system designed to measure and visualize water flow and electrical parameters in real time. The system collects flow rate, voltage, and current data and transmits it to a monitoring dashboard for analysis and observation.

## Features

- Real-time water flow monitoring
- Voltage measurement
- Current measurement
- ESP32-based IoT platform
- Wireless data transmission
- Dashboard visualization
- Data logging support
